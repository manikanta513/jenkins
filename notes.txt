jenkins is a CI/CD tool
it supports 2 types of pipelines 
    . imperative
    . declarative


imperative:

node {
    stage('one') {
        echo 'one'
    }
    stage('two') {
        echo 'two'
    }
}


declarative:

pipeline() {
    agent { any }
    stages {
        stage('one') {
            steps
        }
    }
}
